# Source Documents Integration

## Ціль
Дати можливість створювати документ `Встановлення цін` на підставі інших документів.

## Підтримувані джерела
- supplier_order
- receipt
- supplier_import

## Принцип
Будь-який документ повинен мати можливість віддати список позицій у стандартному форматі:

```
{
  product_id,
  purchase_price,
  currency_id,
  supplier_price,
}
```

## Логіка
1. Користувач відкриває документ (наприклад надходження)
2. Натискає "Встановити ціни"
3. Система викликає:
   create_document_from_source()
4. Створюється документ цін
5. Всі позиції переносяться автоматично
6. Користувач коригує / підтверджує

## Важливо
- логіка не дублюється
- джерело тільки формує дані
- ціноутворення однакове для всіх сценаріїв

## Архітектура
Кожен документ повинен мати метод:
- get_items_for_pricing()

Щоб не було залежностей між модулями.
